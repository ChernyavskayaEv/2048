HOW TO PLAY
Use your arrow keys to move the tiles.
Tiles of the same number merge into one when they touch, the game points are increased by the face value of the resulting tile.
After each move, a new tile with a face value of "2" or "4" appears on the free section of the field, if a tile with a face value of "4", the game points are reduced by 4.
If the location of the tiles or their nominal value does not change when the arrow keys are pressed, then the move is not made.
The game ends with a loss if it is impossible to perform an action after the another move.
The game ends with a win if, when adding tiles on the field, a tile with a face value of "2048" turns out.
